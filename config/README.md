# Configuration

This directory stores configuration files for your tools and integrations.

## Setup

Add your own configuration files here:
- Time tracking tool settings
- API keys (add to `.gitignore`!)
- Project/task ID mappings
- Weekly allocation targets

## Integration with Commands

- **`/time`** reads your time tracking config for project IDs and allocation targets
- **`/status`** reads your config for project mappings and planned hours
- **`/daily`** checks your planning allocations for time tracking health
